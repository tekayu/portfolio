Computer Basics

What is a computer?

A computer is an electronic device that handles data, in a way that can store, retrieve, and process it.

It is made up of hardware and software, hardware being any part of the computer that is a physical structure, and software being essentially a "set of instructions", that tell the computer what it has to do and how it has to do it.

What is the internet

Quoting MDN web docs, "The Internet is the backbone of the Web, the technical infrastructure that makes the Web possible. At its most basic, the Internet is a large network of computers which communicate all together."

GitHub basics

What is github?

At its core GitHub is a version control system, basically a big save button for projects consisting of files and directories.

In modern day programming, using Git is an essential skill as it allows one to not only save files locally but additionally it works as a remote storage device, allowing the public to view files and collaborate freely.

HTML, CSS and JavaScript

What is HTML?

At its core, HTML standing for Hypertext Markup Language is the language that contains all the raw, unformatted data for any webpage. As im typing this right now its being typed into a HTML document.

What is CSS?

CSS on the other hand, standing for Cascading Style Sheets is the determinant to how the web page is styled, changing options such as colour, font styles, font size, and so much more.

What is JavaScript?

And finally JavaScript, the "brain" behind your webpage, allowing manipulation of HTML and CSS elements after the site has already been loaded. This is truly how you make your site feel interactive and alive.
